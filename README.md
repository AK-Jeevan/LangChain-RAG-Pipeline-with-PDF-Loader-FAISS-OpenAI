# LangChain-RAG-Pipeline-with-PDF-Loader-FAISS-OpenAI
A fully commented Retrieval-Augmented Generation (RAG) example built with LangChain. This project loads a PDF, splits it into chunks, generates embeddings, stores them in a FAISS vector index, and builds a retrieval + LLM pipeline using OpenAI. Includes prompt templating, retriever setup, and an end-to-end query example.
